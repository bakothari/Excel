# Excel UI Clone (Frontend Only)

A clean and responsive **Excel-inspired user interface** built using **HTML, CSS, and Vite**.  
This project focuses on replicating the look and feel of a spreadsheet application, including toolbar, ribbon, and grid layout.

---

##  Features

-  Excel-like ribbon (Home, Insert, Page Layout, Formula, etc.)
-  Toolbar with formatting options (font, alignment, styling)
-  Grid-based layout with rows & columns (A–G, 1–10)
-  Sample student data table
-  Dark-themed header UI
-  Responsive layout
